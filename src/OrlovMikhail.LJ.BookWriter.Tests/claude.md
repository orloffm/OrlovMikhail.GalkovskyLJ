# OrlovMikhail.LJ.BookWriter.Tests

Unit tests for the BookWriter library.

## Project Type
- **Type**: Test Library
- **Framework**: NUnit (v2.6.4)
- **Mocking**: RhinoMocks

## Test Coverage

Tests for:
- HTML parsing and tokenization
- Post parts creation
- Processor pipeline transformations
- Text preparation and typography
- Quote handling and chevron processing

## Running Tests

Requires NUnit test runner (v2.x compatible).
